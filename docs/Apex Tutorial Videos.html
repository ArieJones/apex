Apex Tutorial Videos

There are a series of videos that you can use to learn how to get the most out of Apex.

Part 1: Getting Started with the Apex SDK and your first MVVM Application

In this video you will learn:

  • How to install the Apex SDK
  • How to create your first WPF MVVM application
  • How to use Notifying Properties
  • How to use Observable Collections
  • How to create a View

Part 2: Adding Commands to an MVVM Application

Building on part 1, in this video you will learn:

  • How Commands work in MVVM
  • How to add Commands to a View Model
  • How to bind to Commands in a View
  • How to use Command Event Handlers to separate UI code from business logic
