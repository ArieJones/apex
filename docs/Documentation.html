Getting Started with Apex

This page will help you get started with Apex.

Installing the SDK

To use Apex, you must first install the Apex SDK. You can always find a link to the latest release from the main CodePlex Page. Once you have installed the Apex SDK, you will be able to create new applications from the Apex Templates, add Apex classes to existing applications and use the Sample Applications to learn more about how to complete common tasks.
