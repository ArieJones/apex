For Visual Studio 2010

For Visual Studio 2010, to load the Apex Source Code you'll need:

  1. Visual Studio 2010 SP1
  2. Silverlight Tools for Visual Studio 2010 SP1
  3. Windows Phone SDK 7.1
  4. Visual Studio 2010 SP1 SDK
  5. Expression Blend Preview for Silverlight 5 (for Interactivity)

To use the code, load the Apex.VS2010.sln solution file.

For Visual Studio 2012

To use the Apex Source Code, you will need the following tools for Visual Studio:

  1. Visual Studio 2012 SDK
  2. Windows Phone 8 SDK

To use the code, load the Apex.VS2012.sln solution file.

Windows 8 as a Development Environment

On Windows 8, the Windows Phone SDK 7.1 will fail to install fully, however, it will install enough components to allow the Windows Phone projects to be opened.
